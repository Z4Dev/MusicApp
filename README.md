
## Funcionalidades

- Registro/Login
- Recuperação de conta
- Personalização de perfil
- Temas dark e light
- Criação de playlist
- Adicionar a música à playlist
- Ouvir música
- Gerenciar Volume


## Deploy

Para fazer o deploy desse projeto rode

```bash
  yarn run dev
```
